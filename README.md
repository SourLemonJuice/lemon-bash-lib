## 碎碎念

- 这些函数文件开头都会有注释讲解使用方法，不同时期写的注释风格会有些不一样
- 注释第一行应该是该函数没有`llib_`的名称加版本号
  
  ```text
  # function v2
  ```

  这里的版本号在每次更新文件后都可能会改动，并不会统一更改
